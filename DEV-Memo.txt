절대경로/상대경로 차이점 및 사용처 

항목	       절대경로 (Absolute Path)	                상대경로 (Relative Path)
의미	      전체 경로를 처음부터 끝까지 명시      	현재 파일 기준으로 상대적인 위치 표현
형식 예시	       /images/cat.jpg                      ../images/cat.jpg
           http://example.com/images/cat.jpg	      ./assets/img/cat.jpg
시작점     	웹사이트의 루트(/) 또는 도메인 기준	           현재 파일이 위치한 경로 기준
유지보수	사이트 구조 바뀌면 변경 위험 있음           	파일 위치만 알면 쉽게 수정 가능
사용 예시	CDN, 외부 링크, 메인 CSS/JS 연결 시	          프로젝트 내부 리소스 연결할 때
장점	    정확한 위치 지정, 외부 리소스 쉽게 접근         	경로 간단, 이동 편리
단점	        사이트 구조 바뀌면 깨질 위험	       상대적이라 구조 복잡해지면 헷갈릴 수 있음

            상황	                                          추천 경로
외부 리소스 (폰트, 이미지, CDN 등)	                          절대경로 (URL형)
프로젝트 내부 이미지, 파일 등	                                  상대경로
루트 경로 고정 필요 시 (SPA, 프레임워크 등)                    절대경로 (/로 시작)
